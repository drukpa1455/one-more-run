# One More Run submission kit

This directory owns the public story, design handoff, and judging narrative for
the intended converged hackathon build:

- bounded Codex inner edit turns;
- an empirical outer GPU loop that keeps only measured improvements;
- Hindsight graph-backed memory across campaigns;
- Akash GPU execution;
- Pomerium Zero identity-aware access; and
- a content-addressed JSONL evidence trail.

Public copy is intentionally written in the optimistic final tense requested
for the submission. It assumes the integration checklist in
[`integration-review.md`](integration-review.md) is completed before judging.

## Files

- [`devpost.md`](devpost.md): every Devpost field, story, tags, links, media,
  and sponsor-prize rationale.
- [`linkedin.md`](linkedin.md): launch post and short alternate.
- [`carousel.md`](carousel.md): exact eight-page technical carousel and diagram
  instructions, reusable as a judging deck.
- [`judging.md`](judging.md): three-minute talk track, demo path, fallback, and
  technical Q&A.
- [`design-system.md`](design-system.md): complete Opal Light and Garnet Dark
  modes.
- [`carousel_cli.py`](carousel_cli.py): deterministic illustrative CLI frame.
- [`design-agent-handoff.md`](design-agent-handoff.md): one copy-paste prompt for
  the Claude design agent.
- [`integration-review.md`](integration-review.md): private engineering review
  and pre-submission truth gate.

## Canonical thesis

> An agent can change the program. Only an experiment can change the champion.

Use this line once per artifact at most. The closing brand line remains:

> One More Prompt starts the idea. One More Run tests it.

Do not put both lines on the same page.
